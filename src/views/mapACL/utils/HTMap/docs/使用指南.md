# HTMap 使用指南

本文档详细介绍了HTMap地图防腐层的使用方法，包括地图初始化、基础控制、事件处理、视图模式切换等核心功能，为开发者提供完整的使用指导。

## 概述

HTMap 提供了统一的地图操作接口，支持多种地图引擎的无缝切换。通过本指南，您可以快速掌握HTMap的核心功能和使用方法。

## 快速开始

### 1. 地图初始化

#### 基础用法

```javascript
import HTMap from '@/utils/HTMap'

// 创建地图实例
const map = new HTMap('map-container', {
  engine: 'mapbox',
  center: [118.177726746, 39.630245209],
  zoom: 15,
  pitch: 0,
  bearing: 0
})

// 等待地图加载完成
try {
  const mapInstance = await map.init()
  console.log('地图初始化成功:', mapInstance)
  
  // 地图就绪后执行操作
  map.onMapReady(() => {
    console.log('地图已就绪，可以开始操作')
  })
  
} catch (error) {
  console.error('地图初始化失败:', error)
}
```

## 地图控制方法

### 1. 中心点控制

```javascript
// 设置地图中心点
map.setCenter([118.18, 39.64])

// 获取地图中心点
const center = map.getCenter()
console.log('当前中心点:', center)

// 支持字符串格式输入
map.setCenter('[118.18, 39.64]')
```

### 2. 缩放级别控制

```javascript
// 设置缩放级别
map.setZoom(16)

// 获取缩放级别
const zoom = map.getZoom()
console.log('当前缩放级别:', zoom)

// 获取缩放范围
const minZoom = map.getMinZoom()
const maxZoom = map.getMaxZoom()
console.log(`缩放范围: ${minZoom} - ${maxZoom}`)
```

### 3. 倾斜角度控制

```javascript
// 设置倾斜角度
map.setPitch(30)

// 获取倾斜角度
const pitch = map.getPitch()
console.log('当前倾斜角度:', pitch)

// 获取倾斜角度范围
const minPitch = map.getMinPitch()
const maxPitch = map.getMaxPitch()
console.log(`倾斜角度范围: ${minPitch}° - ${maxPitch}°`)
```

### 4. 旋转角度控制

```javascript
// 设置旋转角度
map.setBearing(45)

// 获取旋转角度
const bearing = map.getBearing()
console.log('当前旋转角度:', bearing)
```

### 5. 完整视角控制

```javascript
// 设置完整视角
map.setViewport([118.18, 39.64], 16, {
  pitch: 30,
  bearing: 45
})

// 获取完整视角信息
const viewport = map.getViewport()
console.log('当前视角:', viewport)
// 输出: { center: [118.18, 39.64], zoom: 16, pitch: 30, bearing: 45 }
```

## 常用事件类型

### 基础交互事件

| 事件名称 | 说明 | 事件对象属性 |
|---------|------|-------------|
| `click` | 地图点击 | `latlng`, `point`, `originalEvent` |
| `dblclick` | 地图双击 | `latlng`, `point`, `originalEvent` |
| `contextmenu` | 右键菜单 | `latlng`, `point`, `originalEvent` |

### 视图变化事件

| 事件名称 | 说明 | 事件对象属性 |
|---------|------|-------------|
| `zoom` | 缩放变化 | `zoom`, `type` |
| `zoomstart` | 开始缩放 | `zoom` |
| `zoomend` | 缩放结束 | `zoom` |
| `move` | 地图移动 | `center`, `zoom` |
| `movestart` | 开始移动 | `center` |
| `moveend` | 移动结束 | `center` |

### 视角控制事件

| 事件名称 | 说明 | 事件对象属性 |
|---------|------|-------------|
| `pitch` | 倾斜角度变化 | `pitch` |
| `bearing` | 旋转角度变化 | `bearing` |
| `viewreset` | 视角重置 | `center`, `zoom`, `pitch`, `bearing` |

## 事件处理方法

### 1. 添加事件监听器

```javascript
// 使用 on 方法添加事件监听器
map.on('click', handleMapClick)

function handleMapClick(event) {
  console.log('地图被点击:', event.latlng)
}
```

### 2. 移除事件监听器

```javascript
// 使用 off 方法移除事件监听器
map.off('click', handleMapClick)

// 移除所有点击事件监听器
map.off('click')
```

### 3. 一次性事件监听器

```javascript
// 使用 once 方法添加一次性事件监听器
map.once('zoomend', (event) => {
  console.log('地图缩放完成，这是一次性事件')
})
```

## 完整示例

### 基础地图应用

```javascript
import HTMap from '@/utils/HTMap'

class MapApplication {
  constructor() {
    this.map = null
    this.init()
  }

  async init() {
    try {
      // 创建地图实例
      this.map = new HTMap('map-container', {
        engine: 'mapbox',
        center: [118.177726746, 39.630245209],
        zoom: 15,
        pitch: 0,
        bearing: 0
      })

      // 初始化地图
      await this.map.init()
      
      // 设置地图就绪回调
      this.map.onMapReady(() => {
        this.setupMapControls()
        this.setupEventListeners()
      })
      
    } catch (error) {
      console.error('地图应用初始化失败:', error)
    }
  }

  setupMapControls() {
    // 设置初始视角
    this.map.setViewport([118.18, 39.64], 16, {
      pitch: 30,
      bearing: 0
    })
  }

  setupEventListeners() {
    // 监听地图点击事件
    this.map.on('click', (event) => {
      console.log('地图被点击:', event.latlng)
    })
    
    // 监听地图移动事件
    this.map.on('move', (event) => {
      console.log('地图移动:', event.center)
    })
  }

  // 公共方法：设置地图中心点
  setMapCenter(lng, lat) {
    try {
      this.map.setCenter([lng, lat])
    } catch (error) {
      console.error('设置地图中心点失败:', error.message)
    }
  }

  // 公共方法：设置地图缩放级别
  setMapZoom(zoom) {
    try {
      this.map.setZoom(zoom)
    } catch (error) {
      console.error('设置地图缩放级别失败:', error.message)
    }
  }

  // 公共方法：切换视图模式
  toggleViewMode() {
    const currentMode = this.map.getViewport().pitch > 0 ? '3D' : '2D'
    const newMode = currentMode === '2D' ? '3D' : '2D'
    this.map.setViewMode(newMode)
    console.log(`视图模式已切换到: ${newMode}`)
  }

  // 销毁地图
  destroy() {
    if (this.map) {
      this.map.destroy()
      this.map = null
    }
  }
}

// 使用示例
const app = new MapApplication()

// 在组件销毁时清理资源
// app.destroy()
```

### 响应式地图控制

```javascript
// 响应式地图控制组件
const mapControls = {
  // 中心点控制
  center: {
    lng: 118.177726746,
    lat: 39.630245209
  },
  
  // 缩放控制
  zoom: 15,
  
  // 倾斜角度控制
  pitch: 0,
  
  // 旋转角度控制
  bearing: 0,
  
  // 视图模式
  viewMode: '2D',
  
  // 样式主题
  styleType: 'white'
}

// 监听控制值变化
Object.keys(mapControls).forEach(key => {
  if (key === 'center') {
    // 中心点特殊处理
    watch(() => mapControls.center, (newCenter) => {
      if (map && map.mapInstance) {
        map.setCenter([newCenter.lng, newCenter.lat])
      }
    }, { deep: true })
  } else if (key === 'viewMode') {
    // 视图模式切换
    watch(() => mapControls.viewMode, (newMode) => {
      if (map && map.mapInstance) {
        map.setViewMode(newMode)
      }
    })
  } else if (key === 'styleType') {
    // 样式主题切换
    watch(() => mapControls.styleType, (newType) => {
      if (map && map.mapInstance) {
        map.setStyleType(newType)
      }
    })
  } else {
    // 其他控制方法
    watch(() => mapControls[key], (newValue) => {
      if (map && map.mapInstance) {
        const methodName = `set${key.charAt(0).toUpperCase() + key.slice(1)}`
        if (map[methodName]) {
          map[methodName](newValue)
        }
      }
    })
  }
})
```

## 错误处理

### 1. 参数验证错误

```javascript
try {
  // 经度超出范围
  map.setCenter([200, 39.64])
} catch (error) {
  console.error('设置中心点失败:', error.message)
  // 输出: 设置中心点失败: 经度必须在 -180 到 180 之间
}

try {
  // 纬度超出范围
  map.setCenter([118.18, 100])
} catch (error) {
  console.error('设置中心点失败:', error.message)
  // 输出: 设置中心点失败: 纬度必须在 -90 到 90 之间
}
```

### 2. 格式错误处理

```javascript
try {
  // 格式错误的坐标

  console.error('设置中心点失败:', error.message)
  // 输出: 设置中心点失败: 中心点字符串格式错误，请使用 [经度, 纬度] 格式
}
```

## 注意事项

### 1. 地图初始化
- 在使用地图功能前，必须先调用 `init()` 方法初始化地图
- 地图初始化是异步操作，需要使用 `await` 或 `.then()` 处理

### 2. 事件绑定时机
- 确保在地图初始化完成后再绑定事件
- 使用 `onMapReady` 回调确保地图就绪

### 3. 事件清理
- 在组件销毁时移除事件监听器
- 避免内存泄漏

```javascript
// 在组件销毁时清理事件
onUnmounted(() => {
  if (map) {
    map.off('click')
    map.off('zoom')
    map.off('move')
    map.destroy()
  }
})
```

### 4. 事件性能
- 避免在事件处理函数中执行复杂操作
- 使用防抖或节流处理频繁触发的事件

```javascript
import { debounce } from 'lodash-es'

// 使用防抖处理频繁的移动事件
const debouncedMoveHandler = debounce((event) => {
  console.log('地图移动:', event.center)
}, 100)

map.on('move', debouncedMoveHandler)
```

### 5. 参数验证
- 所有坐标参数都会自动验证格式和范围
- 建议使用 try-catch 包装地图操作方法调用

### 6. 资源清理
- 在页面卸载前，建议调用 `destroy()` 方法清理地图资源

### 7. 引擎差异
- 不同地图引擎对某些参数的支持可能有所不同

## 总结

HTMap 提供了统一、简洁的地图操作接口，让开发者可以轻松地构建功能丰富、交互友好的地图应用。通过合理使用地图控制方法和事件系统，可以实现各种复杂的地图交互功能。

记住关键点：
1. **正确初始化**: 始终等待地图初始化完成后再操作
2. **事件管理**: 在地图就绪后绑定事件，及时清理事件监听器
3. **参数验证**: 所有坐标参数都会自动验证格式和范围
4. **错误处理**: 使用 try-catch 包装地图操作
5. **性能优化**: 使用防抖/节流处理频繁事件
6. **资源管理**: 及时清理不需要的资源
